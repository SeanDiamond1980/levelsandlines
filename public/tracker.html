Setup instructions
1. Go to console.cloud.google.com
2. Create a project → enable Google Sheets API
3. Credentials → Create Credentials → API Key
4. Create a new Google Sheet, name the first tab Goals
5. Share the sheet → Anyone with the link can edit
6. Copy the Sheet ID from the URL (the long string between /d/ and /edit)
7. Paste both above and click Save & connect
